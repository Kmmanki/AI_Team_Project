<br>

----------------

<br>


## preprocessing 모듈
학습시킬 데이터를 전처리 하여 npy 파일로 저장

<br>

----

<br>

## model 모듈
전처리 한 npy 파일을 이용하여 모델을 학습하여 모델을 저장

<br>

----

<br>

## live_to_scale
